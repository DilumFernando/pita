Experiment command reference
============================

Run all commands from the repo root:

  cd /Users/dilumfernando/projects/NETs

Common environment prefix:

  MPLCONFIGDIR=/tmp/mplconfig XDG_CACHE_HOME=/tmp MPLBACKEND=Agg LOKY_MAX_CPU_COUNT=8

Python executable:

  /opt/homebrew/anaconda3/envs/pita-aldp/bin/python
